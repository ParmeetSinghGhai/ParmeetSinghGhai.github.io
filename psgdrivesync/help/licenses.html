The application functionalities remain disabled as long as there is no google drive present in the application so the first step that you should begin with is the addition of your google drive account into this application which is done through "Licenses" window as shown on the right. There is no limit to the number of google drive accounts which can be added to the appplication however there needs to be at least one google drive account present in order for application functionalities like Addition of local drives, Encryption, Filters etc to work.

Please go ahead and click on "Add License" on the left to see how its done. As always thank you for using the application. I Appreciate it.
