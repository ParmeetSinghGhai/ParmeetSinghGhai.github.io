A filtered file can have its linked filters removed from it, new filters added to it and can have existing filters set as Inheritable or not Inheritable. After these kind of changes are made to a filtered file then its needed to be Saved in order for these settings to take effect during sync operations. Following are the steps needed to be performed in order to save filtered files in the application.

1

Select a filtered file in the list of filtered files in the "Filters" window. Doing so will make all its linked filters show up in the panel right next to it. Make all the changes needed to be made to the filtered file like removing a filter, adding a filter, marking a filter as Inheritable or marking a filter as not Inheritable.

1

When you are done making all the changes then you can click on "Save" on the filtered file.

1

The result of the Save operation should show in the status window.
