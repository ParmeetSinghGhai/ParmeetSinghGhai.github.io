Various controls bars presents in different tabs or sections of the application are listed below

1

Licenses - This tab has two controls bars.

  • The Control bar at the top is meant to Save, Delete, Download Existing License and Refresh List of Existing Licenses.
  • The Control bar at the bottom is used to Add a new License, Download Available Application Licenses and Reset the New License Form.

2

Drives - This tab only has one control bar.

  • The Control bar at the top allows user to Mount and Unmount a drive, Add Remote drive, Add Local drive, Save Drive, Delete Drive and Refresh List of Existing Drives

3

Files - This tab only has three control bars.

  • The Control bar with the first option of "Switch to Sync Window" is the browse mode control bar. it allows users to Unmount, Undock and Refresh drive information namely Total space, Used space, Available space, Trash space and User photo linked to the drive.
  • The Control bar with the first option of "Switch to Browse Window" is the sync mode control bar. it allows users to Start Sync operations, Stop Sync operations, Save Sync Containers, Delete Sync Containers and Refresh List of Existing Sync Containers.
  • The Control bar below the sync mode control bar is the control bar that allows users to Save Sync Items, Delete Sync Items and Refresh List of Existing Sync Items.




4

Encryption - This tab only has one control bar.

  • The Control bar allows users to Add Encryption Key, Save Encryption Key, Delete Encryption Key and Refresh List of Existing Keys.

5

Filters - This tab only has two control bars.

  • The Control bar at the top allows users to Add Filters, Save Filters, Delete Filters, Find Files and Folders Linked to a Filter and Refresh List of Existing Filters.
  • The Control bar at the bottom allows users to Save Filtered Files, Delete Filtered Files, Test Filters Attached To Filtered Files and Refresh List of Existing Filters.
