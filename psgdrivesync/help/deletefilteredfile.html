A filtered file can be deleted anytime inside the "Filters" window. Following are the steps needed to be performed in order to delete a filtered file from the filters window.

1

Select a filtered file in the list of filtered files in the "Filters" window. Doing so will make all its linked filters show up in the panel right next to it.

1

Click on "Delete" on the filtered file.

1

Click on "Yes" on the confirmation box.

1

The selected filtered file should not show up anymore in the "Filters" window. The result of the delete operation should also show in system messages.
