An encryption key tag, once created, cannot be edited or changed however the encryption key itself, linked to the encryption key tag, can always be modified and saved. Following are the steps needed to be performed in order to save changes to an encryption key, linked to an encryption key tag.

1

Select the encryption key tag of your choosing in the "Encryption" window.

2

Double click inside the key value box to make the text box editable and then enter a new encryption key value. Press "Enter" key at the end to stop editing the key value.

3

The final step is to save the modified encryption key by clicking on "Save". The same will reflect under system messages.
