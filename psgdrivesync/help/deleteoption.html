All tabs or sections throughout the application (Licenses, Drives, Encryption etc) give out these four ways to delete a data item associated with a list of data items

1

Delete option present on the Control bar.

2

Delete option present in the list of the data item in the same row as the data item with a "Trash Can" icon.

3

Delete option present in right click menu of the data item list itself.

4

Delete option through Keyboard Shortcut Ctrl + X or just the Keyboard key "Delete".
