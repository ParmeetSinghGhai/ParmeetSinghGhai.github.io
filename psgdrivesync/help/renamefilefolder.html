Renaming files and folders works quite the same as it does in any windows operating system. This option is only available in "Files" window browse mode. Following are the steps needed to be performed in order to create a new file or folder

1

Right click on the file and folder that you would like to rename and then click on "Rename" in the right click menu.

2

A popup window should open up next asking for the new name for the selected file or folder. Provide a new name for the file/folder and click on "Rename File" or "Rename Folder".

2

The new file or folder name should reflect on the selected file or folder.
