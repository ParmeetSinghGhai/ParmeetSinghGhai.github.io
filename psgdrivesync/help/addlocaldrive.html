The application has the option to add both remote drives (google drives) and local drives so files and folders can be synced in between them. There is no limit to the number of drives that can be added into the application. Following are the steps needed to be followed in order to add a local drive into the application.

1

Click on "Drives" on the application menu

2

Click on "Add Local Drive" on the control bar or the right click menu on the drives panel

3

A new drive row should populate in the drives list. For a local drive, there is no need to select a license which is why the license drop down list will be greyed out with a text showing "Not Required" however you do need to assign a name to the drive.

The "Add Local Drive" option will scan your computer for all the drives present on it including portable USB drives so you might see multiple drives getting added to the drive list however you do not need to save all of them. You just to save the ones that you need for file and folder syncing.

4

After you have assigned a name to the local drive, you may go ahead and click on "Save Drive" either on the control bar or the right click menu and that should save the drive in the application.

5

Please verify that the drive is saved by reviewing system messages.After the drive is saved then almost all of its properties including the name that you assigned to it will become uneditable. If you would like to change the drive's name then please go ahead and delete this drive and add the same drive with the same license but different name.
