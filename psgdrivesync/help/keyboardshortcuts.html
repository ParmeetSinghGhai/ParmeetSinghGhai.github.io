The application supports the following Keyboard shortcuts

  • Ctrl + S - allows users to Save data items whenever possible.
  • Ctrl + N - allows users to Add New data item(s) whenever possible.
  • Ctrl + A - allows users to Select All data items whenever possible.
  • Ctrl + C - allows users to Copy data items from list of data items.
  • Ctrl + R - allows users to Refresh list of data items.
  • Delete - allows users to Delete data items whenever possible.
  • Ctrl + X - allows users to Delete data items whenever possible.
