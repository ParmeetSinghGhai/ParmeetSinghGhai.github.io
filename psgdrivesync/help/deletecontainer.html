A Sync Container can be deleted anytime however it should not have any Sync Item linked to it otherwise the application will throw a warning messages asking you to delete all the Sync Items linked to the Sync Container first before you delete the Sync Container.
Sync Items can be deleted anytime without any restrictions whatsoever.

  • Following are the steps needed to be performed in order to delete a Sync Container.

    1

    Open the Drive in Sync Mode and Select any one of the Sync Containers from the list of Sync Containers.

    2

    Make sure there are no Sync Items linked to this Sync Container. If they are any then please go ahead and delete those first.

    3

    Use one of the "Delete" options in the Sync Container to delete the Sync Container.

    4

    The Sync Container should no longer show in the list of sync containers and the result of the delete operation should show in system messages as well.




  • Following are the steps needed to be performed in order to delete a Sync Items.

    1

    Open the Drive in Sync Mode and Select any one of the Sync Containers from the list of Sync Containers. Doing so should show up all the Sync Items linked to that Sync Container in the box below. In that list of Sync Items, go ahead and select one of the Sync Items.

    2

    Use one of the "Delete" options for that Sync Item to delete that Sync Item.

    3

    The Sync Item should no longer show in the list of Sync Items and the result of the delete operation should show in system messages as well.
