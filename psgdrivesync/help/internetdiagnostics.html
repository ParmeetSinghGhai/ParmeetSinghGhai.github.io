Internet Connection Diagnostics checks if the application is able to properly access the internet or not. This option can be run from the loading window and the main application window as well. The process involves connecting to www.google.com and verifying if the site is accessible by the application. The following are the steps needed to be performed in order to run internet diagnostics

1

On the loading page, you can click on the "System Diagnostics" tab first.

2

Click on the option for "Diagnose Internet Connection".

3

You can reach the same option from the application main window as well by clicking on the "Help" tab then "System Diagnostics" and "Diagnose Internet Connection".

4

The Results window will then let you know if the application was able to connect to the internet successfully by accessing www.google.com. If it did not, for some reason, then you could try the following to resolve the problem

  • Make sure that your computer is connected to internet either wirelessly or via wired connection.

    More information can be found on
    bleepingcomputer.com and Microsoft's Website


  • Make sure that PSGDriveSync is not being blocked by an Anti Virus software application or a Firewall.
