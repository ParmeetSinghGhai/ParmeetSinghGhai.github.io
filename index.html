PSGDriveSync Application
PSGDriveSync Home Page
PSGDriveSync Github Page
PSGDriveSync application has been build on Dot Net Framework 4.8 and was programmed in C#. It is currently only supported on Windows Operation System. The Application allows users to perform real-time file transfer operations and automated sync operations between

  • Local Drive to Local Drive
  • Local Drive to Google Drive
  • Google Drive to Local Drive
  • Google Drive to Google Drive

The Application also offers AES-256 encryption standard for file and folder encryption along with conditional automated syncing among many other features. Few of the Important Features are listed below


AES-256 Encryption Standard
Uses Advanced Encryption Standard to Encrypt All Your Files and Folders On Local and Remote Drives. Doing so Makes Sure That Your Files and Folders Remain Secure and Protected.


Conditional Automated Syncing
Create Filters Based On Time, Date, File Properties etc to Automatically Filter and Sync Files. You Can Setup "Filters" Based On Current Date & Time, Drive Properties Like Name, Total Space & Available Space on the drive, File Properties like Name, Size, Hash, Last Modified Time of the file, and Folder Properties like Name, Last Modified Time, Type etc of the folder Which Can Help You Sync Only Specific Files and Folders thus Giving You More Control Over The Automated Sync Process.


Real Time File Operations
Perform Real-Time File Operations like Cut, Copy, Paste, Encrypt, Decrypt etc On the Spot. You Can Perform The Following Real Time File Operations On Any One of Your Local and Google Drives AnyTime
  • Create New File & Folder
  • Cut, Copy and Paste File(s) & Folder(s)
  • Delete File(s) & Folder(s)
  • Rename File(s) & Folder(s)
  • Select All File(s) & Folder(s)
  • View File & Folder Properties
  • Encrypt File(s) & Folder(s)
  • Decrypt File(s) & Folder(s)
  • Compare File(s) & Folder(s)
  • Search For Encrypted Files
  • Send Files & Folders to Filters Window


Database Level Encryption
Entire Database Gets Encrypted with AES-256 Standard Encryption With One Master Key That Only You Get To Keep. The Application Saves the Path Of Files and Folders That You Have Set For Automated Syncing In a Local Database along With Your Google Drive's Access Token Used To Access Files From Your Google Drive. In Order to Keep All This Information Safe and Protected, the Application Encrypts the Entire Database Using AES-256 Encryption Standard and a Master Key That Only You Know.
If Needed You can Always Change the Master Key by Using the Existing Master Key.


File Integrity Check For Encrypted File
Use HMAC-SHA256 Hash Check to Simultaneously Verify Both Data Integrity and Authenticity


No Restrictions On Number of Drives
Add as Many Local and Google Drives as You Want. No Restrictions on Number Of Drives That Can be Added To The Application

PSG Password Manager
PSG Password Manager Github Page
PSGPasswordManager application has also been built on Dot Net Framework 4.8 using C# as the programming language. Its currently only supported in Windows Operating System and Requires Dot Net 4.8 Runtime to be installed on the end user's machine. it allow users to
  • Create Secure and Random Passwords upto 2000 Characters Long for One Time Use.
  • Save Those Password in Secure Local Files called 'Vaults' which are themselves encrypted with AES-256 Encryption Standard using a Master Password that only the User knows.

Few of the Important Features are listed below


AES-256 Encryption Based 'Vaults'
Offers AES-256 Encryption Based 'Vaults'. You can save Password(s) or any Other Security Sensitive Data like Bank Account Numbers, Social Security Number etc in these 'Vaults' which are encrypted with AES-256 Encryption Standard. The Master Password used to Encrupt these 'Vaults' is Only know to the End User and is Not Saved Anywhere by the application.


Hidden and System Protected 'Vaults'
All Vault Files are Hidden and System Protected to keep these Files From Getting Accessed by Other Applications. If any one of these Attributes is Changed then the Application will Stop Processing those Vault Files as a Security Measure.


Generate Strong Password
Generate Strong Password of Upto 2000 Characters Long where you get Full Control over how many Lower Case Characters, Upper Case Characters, Special Symbols and Numbers will be present in those Password(s).


Camouflage Your 'Vaults'
The 'Camouflage Vault' Feature under Vault Management tab Allows You to Create upto 100 Random Vaults at a Time in the Same Folder or Directory as your own Vault. This is Done to Make Things Difficult for People Wanting to Decrypt Your Vault Files By Guessing the Master Password that you have used for Your Vaults. Due to the Camouflage feature they would first have to Differentiate between Your Vault File and the "Dummy" Vault Files which Could take Time, Time which you can use To Change all Your Vaults and Password(s).


Automated Typing Of Long Passwords
The "Send To Window" Features Allows You to Get the Application to Type in those Super Long Complex Passwords, that the Application Generates, in Windows Where its Not Allowed to Copy and Paste Password like a Bank's Website thus Saving You From the Trouble of Typing in those Passwords By Yourself.
